# Security Policy

## Scope

This repository contains instruction files, templates, documentation, screenshots, and a packaged ChatGPT skill. It is not a network service and does not process user data.

## Reportable issues

Report these issues:

- malicious or deceptive instruction content
- unsafe guidance that could trigger destructive repository changes
- supply-chain concerns in bundled files
- credential leakage in committed content
- false capability or verification claims

## Not in scope

The following are out of scope unless they create direct safety or integrity risk inside this repository:

- general model behavior debates
- benchmark requests
- feature requests framed as security issues
- unsupported claims about platform behavior outside the files in this repo

## Reporting

Open a private report through GitHub Security Advisories when possible.

Include:

- affected file path
- risk summary
- reproduction steps
- evidence
- suggested containment if known

## Disclosure expectations

Use coordinated disclosure. Do not publish exploit-style writeups against the repository contents before maintainers have time to review and patch.
