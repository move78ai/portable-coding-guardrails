# Portable Coding Guardrails v1.0.0

Repo-first coding guardrails for ChatGPT, Codex, Claude, and Gemini.

## Highlights

- repo-level and project-level control surfaces made primary
- account-global fallback kept intentionally weak
- platform-ready exports for ChatGPT, Codex, Claude, Gemini, and generic chat UIs
- packaged ChatGPT skill in `dist/skill.zip`
- project appendix template for repo commands and invariants
- proof material and launch collateral included

## Why it exists

Short account-global prompts do not carry repo commands, architecture constraints, and local invariants well enough. Those rules belong in repo-local or project-local surfaces.
